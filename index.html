Политика обработки персональных данных
ИП Смановских П.Н.

Версия: 2025
Email: sman.p@mail.ru

1. Общие положения

1.1. Настоящая Политика определяет порядок обработки персональных данных пользователей Telegram-бота, предоставляющего услугу анимации изображений («Сервис»).

1.2. Оператор обрабатывает данные в соответствии с Федеральным законом № 152-ФЗ.

1.3. Использование Сервиса означает согласие пользователя с настоящей Политикой.

2. Какие данные обрабатываются

2.1. Пользовательские данные:

  • изображения, отправленные боту;
  • текстовые сообщения;
  • данные, необходимые для оплаты (email, телефон) при использовании ЮKassa.

2.2. Данные Telegram-аккаунта:

  • username / nickname;
  • ID пользователя;
  • avatar (при наличии);
  • время взаимодействия.

2.3. Технические данные:

  • тип файла, время загрузки;
  • логи ошибок и системные события.

3. Цели обработки данных

  • оказание услуги анимации изображений;
  • передача результата пользователю;
  • обработка платежей через ЮKassa;
  • обеспечение стабильной работы бота;
  • ответы на обращения пользователей.

Данные не используются для идентификации личности. Биометрическая обработка не осуществляется.

4. Правовые основания

  • ст. 6 Федерального закона № 152-ФЗ;
  • согласие пользователя, выраженное использованием Сервиса и нажатием «Старт».

5. Передача данных третьим лицам

5.1. Telegram (Telegram FZ-LLC).

Передача данных происходит автоматически при общении через мессенджер.

5.2. ЮKassa.

Используется для проведения оплаты. ЮKassa — самостоятельный оператор ПД.

5.3. Поставщик анимации изображений (например, Hailuo / PiAPI).

Данные передаются только для выполнения услуги.

Ничего не продаётся, не публикуется, не передаётся третьим сторонам без необходимости.

6. Хранение и удаление данных

  • изображения удаляются после обработки или через 24 часа;
  • логи хранятся до 30 суток;
  • платёжные данные хранятся по правилам ЮKassa.

Удаление всех данных можно запросить по email: sman.p@mail.ru

7. Защита данных

  • ограниченный доступ к системе;
  • защита каналов передачи (HTTPS/TLS);
  • отсутствие публичного хранения;
  • удаление файлов после обработки.

8. Международная передача

Передача данных возможна:

  • в Telegram (ОАЭ);
  • на серверы сервиса обработки изображений.

Передача осуществляется только для оказания услуги.

9. Права пользователя

  • запрос на доступ к данным;
  • изменение, блокировка или удаление;
  • отзыв согласия;
  • жалоба в Роскомнадзор.

10. Изменения Политики

Оператор вправе изменять Политику. Актуальная версия доступна на этой странице.
